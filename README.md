# eslint-configs

Monorepo with eslint configurations.

## Getting started

Install dependencies:

```sh
npm i
```

## Linting

To lint all packages run:

```sh
npx lerna run lint
```

## Publishing

1. Make changes in individual packages and commit changes per usual.
2. Run `npm run release` and follow prompts.
